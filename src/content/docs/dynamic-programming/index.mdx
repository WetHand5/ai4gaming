title: Dynamic Programming & Sequential Decision-Making

本章使用 Markov Decision Process 和 Bellman 方程对不确定性下的序贯决策进行形式化，然后推导出经典的求解方法。

## 推荐阅读顺序

1. MDP — 形式化模型
2. Bellman Equation — 递归结构
3. Value Iteration — 通过价值收敛求解
4. Policy Iteration — 通过策略改进求解
5. Q-Learning — 无模型的价值学习
