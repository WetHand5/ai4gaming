当多个智能体进行策略性交互时，解概念和学习动态与单智能体场景有所不同。本章涵盖均衡概念和多智能体学习方法。

## 推荐阅读顺序

1. Nash Equilibrium — 核心解概念
2. Best Response — 许多算法的基础构件
3. Fictitious Play — 经典学习动态
4. Population-Based Training — 扩展多智能体学习
