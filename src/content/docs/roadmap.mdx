本页概述了本站所涵盖主题的推荐学习路径。

## 学习路径

各章节按照前置依赖关系排序：

1. **Search Algorithms** — 一切的基础，从这里开始。
2. **Monte Carlo Tree Search** — 在搜索的基础上引入采样。
3. **Dynamic Programming** — 形式化序列决策问题。
4. **Reinforcement Learning** — 在 Dynamic Programming 基础上扩展，从交互中学习。
5. **Incomplete Information** — 将 RL 和搜索应用于不完全信息博弈。
6. **Multi-Agent & Game Theory** — 多智能体策略推理。
7. **Hierarchical Planning** — 复杂智能体的实用架构。
