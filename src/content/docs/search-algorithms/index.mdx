title: Search Algorithms
---


搜索算法是博弈和规划中 AI 决策的基础。本章涵盖确定性搜索与对抗性搜索，从经典的博弈树算法到图搜索方法。

## 推荐阅读顺序

0. Hello-Gaming — 写在前面
1. Minimax — 理解对抗性搜索
2. Alpha-Beta Pruning — 使 Minimax 实用化
3. Expectimax — 处理不确定性
4. Iterative Deepening — 时间受限的搜索
5. Transposition Table — 避免重复计算
6. Move Ordering — 提升剪枝效率
7. A* — 启发式图搜索
8. IDA* — 内存受限的 A*
9. Dijkstra — 最短路径基础

## 与其他章节的关系

- MCTS（第 2 章）通过采样而非完全展开来扩展搜索。
- Value Iteration（第 3 章）可视为在具有概率转移的状态空间上的搜索。
- Q-Learning（第 3 章）从经验中学习相同的结构。
